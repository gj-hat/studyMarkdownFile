#### 接下来修改根目录下配置文件_config.yml信息

```
deploy:
type: git
repo: <repository url>
branch: [branch]
message: [message]
```

| 参数    | 描述                                                         |
| :------ | :----------------------------------------------------------- |
| repo    | 库（Repository）地址                                         |
| branch  | 分支名称。如果您使用的是 GitHub 或 GitCafe 的话，程序会尝试自动检测。 |
| message | 自定义提交信息                                               |
