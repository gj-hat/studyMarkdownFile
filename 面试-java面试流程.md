## Java 面试交流流程

1.  开场介绍
    - 简历确认
    - 面试官介绍

2.  面试交流
    - 技术交流
      - Java 基础（线程、锁、BIO、NIO、AIO）
      - 算法
      - 设计模式
      - JVM（JMM、GC、调优）
      - 数据库（基础知识、索引、视图、模型设计、调优、部署）
      - 案例（由面试者现场描述设计思路)
      - 工作职权（工作内容、工作流程、协作方式）
    - 管理交流
      - 项目管理工作内容（团队规模、人员配比、工作职权）
    - 发散性问题交流
      - 最棘手的事情
      - 最有成就感的事情
      - 当前团队现状、工作流程、存在的问题、优秀的点等；
3.  答疑总结
    - 面试人提问、答疑
    - 面试结果反馈周期
    - 薪资
